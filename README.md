# @cosmic-plus/jsutils

Generic JavaScript utilities that are not related to Stellar.

## Install & Import

* **Npm** `npm install @cosmic-plus/jsutils`
* **Yarn** `yarn add @cosmic-plus/jsutils`

Then: `const ${module} = require('@cosmic-plus/base/${module}')`
